Areas
-----
Main -> Projects, Learning/ Tutorials, Register/ Login
Projects -> Datasets, Trees, Analysis, 
Learning 
	-> downloaden der tutorials
	-> Sample anwendung
	-> Introtour mit exercises

Es soll eine Funktionalität zum SI-Contest geben (später)

Detail
------
Learning
	-> Samples, Documents
	-> Aufteilen der Exercises vom SI
	-> Sandbox zum erstellen/ testen eines Baumes
	-> Documents können gedownloaded und dargestellt werden
Projects
	-> es sollen neue Projecte (Welten) erzeugt werden
	-> bestehende können geladen werden
	-> beim erstellen sollen die Attribute erstellt werden
		(die Atts definieren die Welt)
		-> es sollen bestehende Atts gelistet und verwendet werden 
Datasets 
	-> sollen geladen und bearbeitet werden können
	-> können lokal oder vom Server geladen werden
	-> Spalten werden von den Attributen definiert
	-> Zeilen sollen bearbeitet und hinzugefügt werden können
	-> datensätze sollen zu den Projekten gespeichert werden und weiterverwendet werden können
	-> datensätze werden unter verschiedenen ids gelistet
Trees
	-> verarbeiten der bestehenden Atts als Bäume (auch mehrere Bäume sollen erstellt werden können)
	-> besthende äume sollen geldaen bzw gespeichert werden können
	-> anwenden von Datensätze auf bäume
Analysis
	-> auswahl von Bäumen und Datensätzen und diese analyisieren
	-> ergebnisse sollen visuell dargestellt werden
	-> optional erstelle optimale Bäume (C4.4 listen der versch. algs)

Sonstige Anforderungen
	-> erstellung unter web
	-> anwendung soll auch unter mobilen geräten laufen
	-> es soll ein gutes übersichtlichen Theme verwendet werden


Wie?
####

----------------

- Aufschreiben der Spezifikationen für evaldas
- aufsetzen eines servers unter einer public domain
- entwickeln der server-skripte für die clientseite
